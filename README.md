# 📊 Contoso Retail Data Analysis

## 📌 Overview

This is an end-to-end data analytics project using **SQL and Power BI** to analyze customer behavior, retention patterns, and revenue trends based on the Contoso retail dataset.

---

## 🎯 Objectives

* Analyze customer segmentation
* Perform cohort-based retention analysis
* Identify revenue trends and business insights

---

## 🛠️ Tools & Technologies

* SQL (PostgreSQL)
* Power BI *(in progress)*
* Excel (for initial data handling)


## 🔍 Key Analysis Performed

* Customer segmentation (overall and country-level)
* Cohort analysis and retention tracking
* Cohort revenue analysis and YoY cohort impact
* Monthly and yearly revenue trend analysis
* Month-over-month (MoM) sales growth analysis
* Product category performance and contribution analysis
* Top-performing products and category revenue leaders
* Product cross-selling and category pair analysis
* High-value customers and revenue concentration (Pareto analysis)
* Customer purchase behavior (single product, single category, single store)
* Customer inactivity and churn indicators
* Retention rate analysis (monthly cohort retention)
* Median order value and category-level sales insights
* Daily customer trends and regional performance analysis
* Revenue processing time analysis
